## So benutzt du die Seite

### Themen aufklappen
Klick auf eine Themenkarte → alle Prüfungsinhalte, Schlüsselbegriffe und Opinión-Hinweis erscheinen.

### Lernfortschritt tracken
Jede Karte hat eine **„Gelernt"-Checkbox**. Haken setzen → Fortschrittsbalken oben aktualisiert sich automatisch.

### 15-Minuten-Timer
Oben auf der Seite befindet sich ein Übungs-Timer:
- **Start** → Timer läuft rückwärts von 15:00
- **Pause** → Timer anhalten
- **Reset** → zurück auf 15:00

Der Timer wird orange bei < 3 Minuten, rot bei < 1 Minute.

### Schlüsselbegriffe
Unter jedem Thema erscheinen wichtige spanische Fachbegriffe als Tags — ideal für schnelle Wiederholung vor der Prüfung.
